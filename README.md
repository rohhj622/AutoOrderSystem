# AutoOrderSystem

> 자동으로 순서를 정해주는 프로그램입니다.
#eclipse #jsp  #mysql8.0.17 #apacheTomcat9.0.24 #spring-MVC-Modle

## 주요 기능

### 순서 정하는 기능
- 하루에 한번만 가능.
- 같은 날에 한번 더 순서정하기를 클릭 시, 별다른 순서지정없이 내역만 보여주는 기능.

### 학생 추가 기능

### 차트
- 누적된 순서를 보는 기능.
- 만약 누적 값이 가장 클 경우, 자동으로 1번이 되는 기능.

#### +)페널티 기능
당일 취소시 1점씩 부여.
누적 5점이 될시, 취소당일로부터 7일간 예약 불가.
